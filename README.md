# Netflix Roulette UI

## Setup

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app) and uses [Yarn](https://github.com/yarnpkg/yarn) for dependency management.

```
  $ yarn install
```

## Development

```
  $ yarn start
```

Go to `http://localhost:3000/`

## Test

```
  $ yarn run test
```
